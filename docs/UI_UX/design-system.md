# FinPulse UI/UX Design System

## Design Philosophy
FinPulse uses a modern, "glassmorphism" aesthetic suitable for a high-tech financial dashboard. The interface should feel vibrant, responsive, and highly professional.

## Color Palette
- **Backgrounds:** Very dark slate (`#0f172a`), deep charcoal for panels.
- **Accents:** 
  - Brand Blue: `var(--accent-blue)` (e.g., `#3b82f6`) for primary buttons and links.
  - Buy Green: `#10b981` (emerald) for positive trends and BUY buttons.
  - Sell Red: `#ef4444` (red) for negative trends and SELL buttons.
- **Text:** White for primary headers, soft gray (`#94a3b8`) for secondary text.

## Typography
- **Primary Font:** Inter or Roboto (sans-serif). Clean, readable, and monospaced for numerical financial data.
- **Scale:** H1 (32px), H2 (24px), Body (14-16px), Micro (12px for timestamps).

## Components
- **Glass Panels:** Semi-transparent dark backgrounds with a subtle white/gray border and backdrop blur.
- **Inputs:** Darker background than the panels, white text, glowing blue border on focus.
- **Buttons:** Solid colors with subtle hover scale transformations and glow effects.
